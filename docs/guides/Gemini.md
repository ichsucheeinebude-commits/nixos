# 🤖 AI Context: Operational Guides

## Zweck
Jeder Guide beschreibt den Betrieb eines Domains: Prereqs, Data-Flow, Health-Checks, Troubleshooting.

## Regeln
- Template: `_templates/TPL_Guide.md` verwenden
- Keine Nix-Code-Rationale → die steht in der ADR
- Health-Checks müssen konkret sein (Befehle, keine Beschreibungen)
- Troubleshooting-Table mit Symptom → Cause → Fix
